Variable is mechanism for storing data while program is running. In order to create a variable you have to wite:

let myVariable = 12;

Where:

  • let - keyword to create a changeable variable,
  • myVariable - name of the variable,
  • = - assignment operator,
  • 12 - data that will be storing in variable,
  • ; - line end symbol.

In JavaScript you have in your disposal several primitive data types like numbers, strings and booleans.

To set a number, you just need to specify it.

let a = 12;

To set a string as data for storing in variable, you need wrap set of symbols in any type of quotes (single, double or backslash).

let a = "Hello world";

Boolean type need for description of comparison result in the form of true or false.
